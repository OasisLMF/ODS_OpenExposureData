# Open Exposure Data (OED) Relational Model - Beta

This repository contains a **beta implementation** of a relational database model for the [Open Exposure Data (OED) schema](https://github.com/OasisLMF/OpenDataStandards).

## Status

⚠️ **Beta Version - Informational Purposes Only**

This implementation is currently in beta and is provided for **informational purposes only**. It is not yet recommended for production use.

## Implementation

The example provided uses **Microsoft SQL Server** as the target database platform. The schema structure demonstrates one approach to implementing OED specifications in a relational format.

## Usage

This repository is intended to:
- Demonstrate a relational approach to OED data modeling
- Provide a reference implementation for discussion and feedback
- Support exploration of OED schema capabilities in a database context

## Feedback

Comments, suggestions, and contributions are welcome as we refine this implementation toward a production-ready release.
