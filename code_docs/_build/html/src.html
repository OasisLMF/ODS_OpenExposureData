src.oed.read_csv(filepath_or_buffer, df_engine=<module 'pandas' from '/Users/maxwellflitton/Documents/oasis/github/OpenDataStandards/venv/lib/python3.8/site-packages/pandas/__init__.py'>, file_type=None, dtype=None, default=None, **kwargs)[source]¶

the function read_csv will load a csv file as a DataFrame with all the columns converted to the correct dtype and having the correct default. both dtype and default can be overwritten if needed it will also try to save space by converting string dtype into categories

By default, it uses pandas to create the DataFrame. In that case you will need to have pandas installed. You can use other options such as Dask or modin using the parameter df_engine.

Parameters:

  • filepath_or_buffer – any path or file like object accepted by your df_engine (for pandas see pandas.read_csv)

  • df_engine – engine that will convert csv to a dataframe object (default to pandas if installed)

  • file_type – [Loc, Acc, ReinsScope, ReinsInfo]

  • dtype – dtype dictionary that will overwrite the default dtype mapping

  • default – dictionary of default value to apply to overwrite the default Value mapping

  • kwargs – extra argument that will be passed to the df_engine

Returns:

df_engine dataframe
